Register Your Kid

December 2018-February 2019

Register Your Kid is a Ruby on Rails and JavaScript project I made that allows parents to register their kids for art classes in a school. This was inspired by a non-profit arts center in my community which runs great classes that my family attends, and I hope it can someday be part of their actual registration process to make it easier for the diverse families they serve. I used MVC programming concepts to amke the backend of this come alive, and added JavaScript front end functionality using jQuery and AJAX API calls. Check out the code here

Jaime Andrews Fitness Training

January 2018

My first freelance project was this website, although it was recently taken down by my client. Jaime Andrews needed a simple wordpress site for her health coaching clients to see what she offers. I was proud to help her start fresh and create a thriving business. She provided me with some design direction and images/content, and I used Wordpress for the first time to customize a pre-made theme. I used only HTML & CSS here because that was all she required, though I am capable of more. I learned how to make a CSS hamburger navigation menu in order to spice it up a bit. This took me about 20 hours total to complete, including contact time with GoDaddy to make a subdomain and then make the site live.
